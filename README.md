# Video downloading & combine with external audio

## Scripts

- `bin/` - Scripts for video processing
  - `make` - Main processing script
  - `video` - Video downloading tool

## Usage

Run the make script to get video and combine with an audio:

```
./bin/make YOUTUBE_URL AUDIO_URL
```

- `current/` - Working directory for current video processing
- `videos/` - Collection of results videos
